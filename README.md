# Hackathon_Terra_Signal
Materials for the Hackathon of a fake Telco Company named Terra Signal. Includes the dataset, and a starter notebook.

Important Files:
- RENAME_ME.ipynb: A notebook to get you started
- train.csv: The training dataset. You must build a model to predict the 'Churn' column.
- test.csv: The test dataset. The 'Churn' column is missing. You must use the model you built to predict the outcome of each row and save it to 'prediction.csv' as shown in the notebook provided.
